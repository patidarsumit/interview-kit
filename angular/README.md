# Angular Interview Kit

This folder covers Angular topic by topic, from older Angular concepts to the latest Angular v21 features.

Angular is too large for one readable file, so every major topic has its own markdown file under [topics](./topics).

Coding examples are in [programs](./programs).

## Senior Preparation Rule

For every topic, practice explaining:

- what problem it solves
- how it worked in older Angular
- how modern Angular v21 recommends doing it
- common mistakes
- performance and testing impact
- what you would choose in a production application
